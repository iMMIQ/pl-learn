# PL Learn - Programming Language Theory Interactive Course

Learn Programming Language theory through hands-on implementation in OCaml.

## Getting Started

```bash
# Install dependencies
opam install dune menhir alcotest ocaml-lalr

# Build
make

# Run tests
make test

# Start REPL
make repl
```
